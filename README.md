## 1. To do list


To do list를 직접 만들어보는 프로젝트 입니다.

Html + Css + Javascript을 사용하였으며, 추가 버튼을 누를시 다음과 같이 항목을 적고 추가 버튼을 누르게 되면 밑에 내용이 추가되는 것을 확인 하실 수 있습니다. 추가 된 항목을 다시 누르게 되면 밑줄로 해당 항목을 처리 했다는 것을 보기 쉽게 표현 하였습니다.
